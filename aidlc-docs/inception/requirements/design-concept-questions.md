# Design Concept Questions

Нужно выбрать визуальную концепцию для проекта.
Все три варианта — dark cinematic seinen, просто с разным настроением.

---

## Краткое описание концепций

### "Titan" (Атака Титанов)
- **Палитра**: тёмно-серый (#0F0F0F, #1A1A1A), тускло-коричневый/охра (#8B6914), белый текст
- **Настроение**: военный архив, выжженная бумага, тяжёлая статика, мрачная серьёзность
- **Типографика**: sans-serif с засечками в заголовках (Bebas Neue / DM Serif), плотная вёрстка
- **UI-акценты**: узкие полосы-разделители, метки как в военных досье, минимум декора
- **Ощущение**: использовать как тяжёлый рабочий инструмент — без излишеств, строго

### "Chainsaw" (Человек-бензопила)
- **Палитра**: чёрный (#0A0A0A), кроваво-красный (#C0392B, #E74C3C), грязно-белый
- **Настроение**: жёсткий панк-журнал, высокий контраст, агрессивная энергия
- **Типографика**: рубленый bold sans-serif (Oswald / Anton), неожиданные перекосы в акцентах
- **UI-акценты**: Aceternity UI — spotlight эффекты, glitch-hover, резкие тени
- **Ощущение**: визуально захватывает, но может утомлять при долгом использовании ежедневно

### "Naruto grown-up" (зрелый Наруто)
- **Палитра**: deep navy (#0D1117), тёмный индиго (#1E2A3A), янтарный/оранжевый акцент (#E67E22)
- **Настроение**: ночное небо, восточная элегантность, сдержанная теплота
- **Типографика**: современный sans-serif (Inter / Geist), лёгкие геометрические акценты
- **UI-акценты**: мягкое свечение на карточках, плавные анимации Aceternity, warm glow
- **Ощущение**: самый "уютный" из трёх — кинематографично, но комфортно смотреть часами
